So Apple Podcasts sucks. The only way I've been able to get top podcasts is if you have the Apple Podcasts app on a Mac computer and that app is synced with any device you listen to podcasts on. Then follow these steps (it's pretty straight forward): Open Finder. At the top toolbar click Go -> Go to Folder. Type in "~/Library/Group Containers" and press Go. One of the folders should end with "apple.podcasts". Open that, then open the "Documents" folder. Then drag and drop the file called "MTLibrary.sqlite" into where it says "Choose File" above.
